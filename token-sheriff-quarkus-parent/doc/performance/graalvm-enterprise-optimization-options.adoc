== Overview

GraalVM Enterprise Edition (Oracle GraalVM) specific optimization options. These features require Oracle GraalVM or commercial licensing.

**Context**: Token-Sheriff targets GraalVM Community Edition (Mandrel) as primary runtime +
**Current Performance**: Community Edition achieves 25,178 ops/s JWT validation (March 2026)

== Enterprise Edition Features

=== Profile-Guided Optimization (PGO)

[source,bash]
----
# Step 1: Build instrumented executable
native-image --pgo-instrument YourApp

# Step 2: Run with representative workload to collect profile data

# Step 3: Build optimized executable
native-image --pgo=default.iprof -O3 YourApp
----

* 15-20% performance improvement (Oracle estimates, not verified)
* 10-15% smaller binary size (Oracle estimates, not verified)

=== G1 Garbage Collector

[source,bash]
----
native-image --gc=G1 YourApp
----

* Linux only with Oracle GraalVM
* Improved latency consistency for memory-intensive workloads
* Serial GC (default) performs well for JWT validation workloads

=== Jipher JCE Provider

WARNING: Jipher is **no longer bundled** with Oracle GraalVM 25. Must be downloaded separately.

Relevant only for FIPS 140-2 compliance, not performance optimization.

=== ML-Powered Optimization (GraalVM 25)

Automatically enabled with `-O2` and `-O3`. XGBoost-based static profiler providing 1-3% boost. Now available in Community Edition (Mandrel) as well.

== Build Configuration Reference

=== Community Edition (Mandrel)

[source,bash]
----
native-image -O2 \
  --enable-monitoring=jcmd,jfr,heapdump \
  --enable-url-protocols=https \
  --enable-http --enable-https \
  YourApp
----

NOTE: `--enable-all-security-services` was deprecated in GraalVM 24 and removed in GraalVM 25 -- all security services are now enabled by default. This project does not use the flag; no backward compatibility configuration is needed.

=== Enterprise Edition (Oracle GraalVM)

[source,bash]
----
native-image -O3 \
  --pgo=production.iprof \
  --gc=G1 \
  --enable-monitoring=jcmd,jfr,heapdump \
  --enable-url-protocols=https \
  --enable-http --enable-https \
  -march=native \
  YourApp
----
