== Overview

Bibliography for the `commons` base-layer capability (the `de.cuioss.sheriff.token.commons.*`
packages inside `token-sheriff-validation` — see xref:architecture.adoc[Architecture]).

Commons owns four cross-cutting concerns — *IdP transport*, the *error model*, *security
events* and *metrics* — so its sources are the HTTP/OAuth/OIDC transport standards plus
the HTTP problem-details standard. Standards already carried by the validation
bibliography (JWK/JWKS, OAuth 2.0 core, the Security BCP) are *cited here, not restated*;
follow the xref:../validation/requirements.adoc[validation requirements] for their full
entries.
